Ziel

Interaktive D&D-Kampagnen-Website mit:

Frontend: SvelteKit

Styling: TailwindCSS + Typography Plugin

Map: MapLibre GL (https://maplibre.org/maplibre-gl-js/docs/)

CMS/Backend: PocketBase (läuft auf derselben Website / Domain)

Versionierung: GitHub

Layout: Bento-Boxen im Grid mit variabler Höhe (grid-row)
der grit soll so in ein div eingebettet werden das er je nach bildschirm größe links von der map ist oder darunter.
text in rem angeben mit clamp so das der text auf größeren bildschirmen gröser wird bis zu einem gewissen minimum.
